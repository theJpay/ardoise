# Contributing

Thanks for the interest in Ardoise.

This is a personal project, shared as a portfolio artifact and for friends who want to use it. **I'm not accepting pull requests** — any PRs opened will be closed without review.

## What I do welcome

- **Bug reports** — if something is broken, [open an issue](https://github.com/theJpay/ardoise/issues/new) with steps to reproduce.
- **Feature suggestions** — if you'd use something that doesn't exist yet, [open an issue](https://github.com/theJpay/ardoise/issues/new) describing the use case. No guarantee I'll build it, but good ideas shape the roadmap.

## What you can do with the code

Ardoise is MIT-licensed. You're free to fork, modify, and build your own thing on top of it. If you do, I'd love to hear about it — but it's not required.
